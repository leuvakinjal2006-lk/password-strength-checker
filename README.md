🔐 Password Strength Checker

📖 About the Project

In today's digital world, using a strong password is one of the most important steps toward protecting personal information and online accounts. The Password Strength Checker is a simple yet effective web application that helps users evaluate the security of their passwords in real time.

The application analyzes different aspects of a password, including its length, use of uppercase letters, numbers, and special characters. Based on these checks, it categorizes the password as Weak, Medium, or Strong and provides feedback to help users create more secure passwords.

---

🎯 Project Objective

The main objective of this project is to promote password security awareness and demonstrate how front-end technologies can be used to perform real-time password validation. It also helps users understand the characteristics of a strong and secure password.

---

🚀 Key Features

- Real-time password strength analysis
- Classification of passwords into Weak, Medium, and Strong categories
- Detection of uppercase letters, numbers, and special characters
- Instant feedback for better password creation
- Simple and user-friendly interface
- Responsive design for different screen sizes

---

⚙️ How to Use

1. Open the application in your web browser.
2. Enter a password in the input field.
3. The system will automatically analyze the password.
4. View the strength result displayed on the screen.
5. Improve the password based on the provided feedback.

---

🔍 Password Evaluation Criteria

The application evaluates passwords using the following parameters:

- Minimum length requirement
- Presence of uppercase letters (A–Z)
- Presence of numeric digits (0–9)
- Presence of special characters (!@#$%^&*)

Strength Categories

- 🔴 Weak
- 🟡 Medium
- 🟢 Strong
